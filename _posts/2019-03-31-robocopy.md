## 複製

{% highlight powershell %}
robocopy source destination /E /ZB /COPYALL [/LOG:1.txt] /R:3 /W:3
{% endhighlight %}

## 同步

{% highlight powershell %}
robocopy source destination /E /B /COPYALL /MT:64 [/LOG:1.txt] /R:3 /W:3 /MIR
{% endhighlight %}
